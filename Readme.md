# 🎙️ Sermon Shorts Generator  
## 설교 숏폼 자동 생성기

긴 설교 영상을 입력하면, AI가 핵심 하이라이트를 분석하고  
자막이 포함된 **1분 내외의 세로형 숏폼 영상**으로 자동 변환해주는 개인용 자동화 도구입니다.

이 프로젝트의 목표는 단순히 긴 영상을 잘라내는 것이 아니라,  
**설교의 핵심 메시지가 살아 있는 쇼츠형 콘텐츠**를 빠르게 제작할 수 있도록 돕는 것입니다.

---

# ✨ 주요 기능 (Key Features)

## 1. YouTube 영상 입력 지원
- 유튜브 URL만 입력하면 영상 또는 오디오를 자동으로 가져옵니다.
- 설교 전체 영상을 따로 수동 다운로드하지 않아도 됩니다.

## 2. 음성 인식 기반 스크립트 추출
- OpenAI Speech-to-Text를 사용하여 설교 음성을 텍스트로 변환합니다.
- 자막 생성에 필요한 타임스탬프 정보도 함께 추출할 수 있습니다.

## 3. AI 하이라이트 분석
- 전체 설교 스크립트를 AI가 분석하여 숏폼으로 적합한 핵심 구간을 선정합니다.
- 단순히 임의의 60초를 자르는 것이 아니라, 다음 기준을 반영합니다:
  - 첫 부분의 후킹 가능성
  - 문맥 독립성
  - 감정적 강조
  - 메시지 응집도
  - 짧은 영상으로 전달 가능한 완결성

## 4. 세로형 숏폼 자동 변환
- 16:9 가로 영상을 9:16 세로 비율로 변환합니다.
- 기본적으로 중앙 기준 세로 크롭을 적용하며,
  추후 인물 중심 리프레임 기능으로 확장할 수 있습니다.

## 5. 자동 자막 생성 및 합성
- 추출된 자막을 영상 하단에 자동 합성합니다.
- 숏폼에 맞는 큰 글씨, 높은 가독성, 안전한 배치 영역을 고려합니다.

## 6. 최종 숏폼 영상 출력
- 자막이 포함된 완성형 `.mp4` 숏폼 파일을 저장합니다.
- 유튜브 쇼츠, 인스타 릴스, 틱톡 등에 활용 가능한 형태를 목표로 합니다.

|------|---------------|------|

# 🎯 목표 퀄리티

이 프로젝트는 단순한 자동 요약 영상이 아니라,  
다음과 같은 요소를 갖춘 **설교 쇼츠형 결과물**을 목표로 합니다.

- 첫 1~3초 안에 핵심 메시지가 드러나는 구성
- 큰 자막과 높은 가독성
- 설교자의 얼굴 또는 상반신이 안정적으로 보이는 세로 구도
- 또렷한 음성 전달
- 짧지만 메시지가 완결된 하이라이트 구간
- 반복 제작이 가능한 템플릿 기반 편집 구조

즉, “긴 설교를 짧게 줄인 영상”이 아니라  
**처음부터 숏폼처럼 소비될 수 있는 결과물**을 만드는 것이 목표입니다.

# 🚀 전체 워크플로우 (Workflow)

## Step 1. Input
사용자가 유튜브 설교 영상 URL을 입력합니다.

## Step 2. Extract
- `yt-dlp`로 영상 또는 오디오를 다운로드합니다.
- 오디오를 STT 처리 가능한 형식으로 추출합니다.

## Step 3. Transcribe
- OpenAI Speech-to-Text를 이용해 설교 음성을 텍스트로 변환합니다.
- 필요 시 자막용 타임스탬프와 세그먼트 정보도 함께 생성합니다.

## Step 4. Analyze
- 생성된 전체 스크립트를 Gemini API에 전달합니다.
- AI는 숏폼으로 적합한 구간을 분석하여
  시작 시간 / 종료 시간 / 추천 제목 등을 반환합니다.

## Step 5. Edit
- 선택된 구간만 잘라냅니다.
- 영상을 9:16 세로 비율로 변환합니다.
- 자막을 생성 및 정리한 뒤 영상에 합성합니다.

## Step 6. Output
- 최종 결과물을 `.mp4` 파일로 저장합니다.

# 🧠 하이라이트 선정 기준

이 프로젝트에서 가장 중요한 품질 요소는  
“어떤 60초를 선택하느냐”입니다.

따라서 하이라이트 선정 시 아래 기준을 사용합니다.

## 우선 고려 요소
1. **첫 문장 후킹력**
   - 처음 1~3초 안에 시청자를 붙잡을 수 있는가

2. **문맥 독립성**
   - 앞뒤 설명이 없어도 의미가 전달되는가

3. **감정 고조**
   - 설교의 강조, 호소, 선포, 질문 등이 살아 있는가

4. **메시지 응집도**
   - 하나의 주제가 짧은 시간 안에 명확히 전달되는가

5. **숏폼 적합성**
   - 짧은 영상으로도 여운과 전달력이 있는가

## 피해야 할 구간
- 맥락 없이는 이해가 어려운 부분
- 설명은 길지만 임팩트가 약한 부분
- 호흡이 너무 느린 부분
- 예화 도입만 있고 결론이 없는 부분
- 자막으로 옮겼을 때 지나치게 긴 문장 위주의 부분

# 📝 자막 처리 원칙

설교 쇼츠에서 자막은 단순한 보조 기능이 아니라  
영상의 핵심 전달 수단입니다.

## 자막 기본 원칙
- 한 번에 너무 많은 텍스트를 보여주지 않는다
- 문장 전체보다 의미 단위로 짧게 끊는다
- 모바일 화면에서 읽기 쉬운 크기를 유지한다
- 하단 영역에 안정적으로 배치한다
- 설교자의 얼굴과 겹치지 않도록 한다

## 권장 자막 스타일
- 큰 흰색 볼드 텍스트
- 검은 반투명 배경 또는 외곽선
- 1~2줄 제한
- 한 줄 길이 최소화
- 핵심 단어는 강조 가능

## 자막 후처리 필요성
STT 결과를 그대로 자막으로 넣으면
문장이 너무 길고 읽기 어려운 경우가 많습니다.

따라서 실제 출력 전에는 아래 작업이 필요합니다.
- 문장 축약
- 줄바꿈 최적화
- 불필요한 군더더기 제거
- 강조 구간 분리
- 숏폼용 리듬감 조정

# 📐 영상 편집 원칙

## 화면 비율
- 입력: 주로 16:9 가로 영상
- 출력: 9:16 세로 영상

## 기본 크롭 방식
- 중앙 기준 세로 크롭
- 설교자 중심 구도 유지
- 하단 자막 영역 확보

## 향후 고도화 가능 요소
- 얼굴 인식 기반 자동 리프레임
- 장면 전환 감지
- 상단 제목 자동 삽입
- 교회 로고 및 워터마크 삽입
- 배경 블러 활용 세로 확장 레이아웃
